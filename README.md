# MakerBotStatus

##### Requirements
* Python 2.7 (https://www.python.org/downloads/)
* makerbot-gen5-api (https://github.com/gryphius/makerbot-gen5-api)

##### Running
* python makerbot.py <ip address> [auth code] [web server]
